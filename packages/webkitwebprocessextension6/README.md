# D binding for the WebKit 6 library - WebKit web extension and DOM library

This [Dub](https://dub.pm/) sub-package of [giD](https://gid.dub.pm) provides a [D language](https://www.dlang.org) binding to the [WebKitWebProcessExtension 6.0](https://webkit.org) library.
